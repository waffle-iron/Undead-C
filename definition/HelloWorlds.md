# Hello World Programs

## Procedural Program
```
QUICK REVIVE
    SCREAM("Hello World!");
DIE MAGGOT
```

## OOP Program
```
TAKE A TUG OF helloClass {
    REVIVE sayHello(name) {
        SCREAM("Hello " . name);
    }
}

QUICK REVIVE
    BEING instance = SPAWN(helloClass);
    BEING name = SPLENDIFEROUS();
    instance->sayHello(name);
DIE MAGGOT
```
